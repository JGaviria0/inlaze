  
  

## Installation

  

```bash

$  npm  install

```

  

## Running the app

  

```bash

# development

$  npm  run  start

  

# watch mode

$  npm  run  start:dev

  

# production mode

$  npm  run  start:prod

```

  

## Documentation

  
```bash
$ npm run start
```

http://localhost:3000/documentation

  

## Test

  

```bash

# unit tests

$  npm  run  test

  

# e2e tests

$  npm  run  test:e2e

  

# test coverage

$  npm  run  test:cov

```
